E-Commerce Sample Dataset
=========================

Files included:
 - customers.csv
 - products.csv
 - orders.csv
 - order_items.csv
 - ecommerce_sample.sql  (MySQL-compatible dump)

Usage:
 - You can load the CSVs into pandas or import the SQL dump into a MySQL server.
 - The data spans ~1 year and includes 50 customers, 30 products, 200 orders.
